# How to get started

## Software you'll need
- Hugo
  - tested: hugo v0.89.4+extended
- Node
  - tested: Node 16.13.0 / npm 8.1.0
- Yarn
  - tested: 1.22.11

## Let's start coding
- Open 2 Terminals/Consoles in the checkout out folder
    (yeah, 2...)
- First terminal:
  - `yarn install`
  - `yarn watch`
- Second terminal:
  - `hugo server -D`
